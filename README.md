# Loadsheet-autonome

Outil de gestion de chargement autonome pour la logistique.

## Structure du projet

```
src/           # Code source principal
tests/         # Tests unitaires et d'intégration
data/          # Données de référence (gabarits, limites, etc.)
docs/          # Documentation
```

## Installation

```bash
pip install -r requirements.txt
```

## Utilisation

```bash
python -m src.main
```
